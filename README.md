# Analysis of Hospital Appointments

This project provides an analysis of hospital appointments, including the total appointments, attendance rate, and appointment distribution across various departments.

## Key Insights

1. **Total Appointments**: The total number of appointments is 300, with 208 (69.33%) attended and 92 (30.67%) missed.
2. **Attendance Rate**: The overall attendance rate is 0.69, indicating that 69% of appointments were attended.
3. **Appointment Distribution by Department**: The departments with the highest number of appointments are General Medicine (64), Cardiology (57), and Pediatrics (48).
4. **Gender Distribution**: The appointment distribution is relatively balanced between male and female patients.
5. **Appointments by Year, Quarter, Month, and Day**: The graph shows the fluctuations in the total number of appointments over time, with higher appointment volumes during certain periods.

## Visualizations

The project includes the following visualizations:

1. **Pie Chart**: Showing the breakdown of attended, cancelled, and no-show appointments.
2. **Bar Chart**: Displaying the appointment distribution by department.
3. **Line Chart**: Illustrating the total appointments by year, quarter, month, and day.
4. **Bar Chart**: Comparing the number of male and female patients.

These visualizations provide a comprehensive overview of the hospital's appointment data and can be used to identify trends, optimize resource allocation, and improve patient experience.

This project can be used by hospital administrators, data analysts, and healthcare professionals to gain insights into the hospital's appointment management and make informed decisions.
